# Mentoring Bharat

A professional networking platform for IIT-IIM-NIT alumni belonging to the Constitutionally marginalized Scheduled Castes and Scheduled Tribes.

## About

Mentoring Bharat aims to create a supportive community for alumni from premier institutions to connect, collaborate, and mentor.
